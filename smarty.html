Иногда требуется определить последний элемент в выводе цикла Smarty, чтобы присвоить ему какой-нибудь особенный класс или, к примеру, не ставить после него запятую. Сделать это можно так:


{foreach from=$article.TAGS item=tag name=tart}
    <a href="#{$tag.PP_TAG_ID}">{$tag.TITLE}</a>{if !$smarty.foreach.tart.last}, {/if}
{/foreach}

Иногда требуется с помощью шаблонизатора Smarty создать элемент после определенного повтора foreach. Например, после каждого четвертого элемента вывода, при этом он не должен быть последним, вставлять див с чертой. Делается это вот так:


{if $smarty.foreach.3new.iteration %4 == 0 && !$smarty.foreach.3new.last} 
    <div class="main_news_line"></div>
{/if}

Пример адресной строки: http:///mysite.ru/?sort_by=status

{$smarty.get.sort_by}
