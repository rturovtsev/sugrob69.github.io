100% рабочий и простой способ на CSS как прижать футер к низу окна браузера.
Footer прижимается вниз за счет вытягивания блока wrapper на всю высоту окна браузера и подъема футера вверх через отрицательный отступ для избавления от появляющегося при этом вертикального скролла. В данном случае нужно обязательно указывать высоту подвала. Внутри блока wrapper нужно разместить блок push с высотой, равной высоте подвала. Это необходимо для того, чтобы footer не наползал на контент при уменьшении окна браузера.

Код HTML:


<div class="wrapper">
    <div class="header"></div>
    <div class="content"></div>
    <div class="push"></div>
</div>
<div class="footer"></div>

Код CSS:


* {
    margin: 0;
    padding: 0;
}
html, body {
    height: 100%;
}
.wrapper {
    min-height: 100%;
}

.header {
    height: 190px;
}
.footer, .push {
    height: 140px;
}

.footer {
    height: 140px;
    margin-top:-140px;
}

Краткая информация


Правило @media позволяет указать тип носителя, для которого будет применяться указанный стиль. В качестве типов выступают различные устройства, например, принтер, КПК, монитор и др.

Тип Описание
all Все типы. Это значение используется по умолчанию.
aural Речевые синтезаторы, а также программы для воспроизведения текста вслух. Сюда, например, можно отнести речевые браузеры.
braille Устройства, основанные на системе Брайля, которые предназначены для слепых людей.
handheld Наладонные компьютеры и аналогичные им аппараты.
print Печатающие устройства вроде принтера.
projection Проектор.
screen Экран монитора.
tv Телевизор.

Синтаксис


@media тип1 [, тип2] {
  Описание стиля
}

После ключевого слова @media идет один или несколько типов носителя, перечисленных в табл. 1; если их больше одного, то они разделяются между собой запятой. После чего следуют обязательные фигурные скобки, внутри которых идет обычное описание стилевых правил.


Пример задания стилей под определенные размеры экрана браузера

Синтаксис под определенную ширину экрана:


@media screen and (min-width: ...px) and (max-width: ...px) {
  Описание стиля, который будет применяться только при размере экрана, заданном в правиле
}

Иногда требуется сделать на сайте какие-нибудь геометрические фигуры. Быстрее и правильнее делать их с помощью CSS. Если в будущем придется, к примеру, поменять их цвет, то фигуры, сделланные картинками, придется перерисовывать. Фигуры, сделанные с помощью CSS, изменяются секундными правками файла стилей.
